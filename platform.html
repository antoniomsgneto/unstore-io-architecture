Technology Stack

LayerTechnologyPurpose
Frontend Astro SSG TypeScript Static site generation — 5 page types, 5,000+ pages at build, <200KB per page, Lighthouse 95+
Backend API FastAPI Python SQLAlchemy async asyncpg Public REST API, admin endpoints, SSG data feeds, download redirect (302)
Database Supabase PostgreSQL Alembic Primary data store — apps, versions, categories, scans, snapshots. Managed migrations.
Object Storage Cloudflare R2 Binaries (PC), icons (WebP 256x256), screenshots (WebP max 1200px). CDN at cdn.unstore.io
Search Meilisearch Full-text search index — name, developer, description. Fallback to PostgreSQL ILIKE
Cache Upstash Redis API response cache — 5min app detail TTL, 1min trending TTL
PC Scraper httpx async BeautifulSoup4 lxml rapidfuzz Crawl Softonic + Uptodown, extract metadata, dedup, download binaries
Android Pipeline httpx async Aptoide internal API ingestion, category mapping, CDN link management
Content Pipeline GPT-4o-mini / Claude Haiku LLM rewriting (200-400 words), quality checks, ~$0.01-0.02 per app
Security VirusTotal API v3 Hash-first scan, file upload fallback. 4 req/min free tier. Blocks flagged downloads.
Frontend Hosting Cloudflare Pages Static site deployment at unstore.io, global CDN, Full Strict SSL
API Hosting Railway / Fly.io FastAPI backend at api.unstore.io
CI/CD GitHub Actions Daily build 03:00 UTC + webhook trigger. Target <10min for 5,000 pages
Monetisation Google AdSense Single ad unit on download interstitial page only
Monitoring Sentry UptimeRobot GSC Cloudflare Analytics Error tracking, 5-min uptime checks, search indexing, privacy-friendly analytics

Key design decisions: Fully static frontend (no SSR) for maximum performance and minimal hosting cost. Android APKs are never re-hosted — download links point to Aptoide CDN. Download stats come from scraping, not from tracking user downloads (no download_events table). Single AdSense unit on download interstitial only — no ads on content pages. All binaries are VirusTotal-scanned; flagged binaries are blocked at the API level (403).
